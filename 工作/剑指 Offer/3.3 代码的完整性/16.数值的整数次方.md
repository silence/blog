实现 pow(x, n) ，即计算 x 的 n 次幂函数（即，x^n^）。不得使用库函数，同时不需要考虑大数问题。

```
示例 1：

输入：x = 2.00000, n = 10
输出：1024.00000
```

```
示例 2：

输入：x = 2.10000, n = 3
输出：9.26100
```

```
示例 3：

输入：x = 2.00000, n = -2
输出：0.25000
解释：2-2 = 1/22 = 1/4 = 0.25
```


提示：

- -100.0 < x < 100.0
- -231 <= n <= 231-1
- -104 <= xn <= 104

---

方法

二分法求幂

- 当 n 为偶数：x^n^ = (x^2^)^n/2^
- 当 n 为奇数：x^n^ = x·(x^2^)^(n-1)/2^

```javascript
/**
 * @param {number} x
 * @param {number} n
 * @return {number}
 */
var myPow = function(x, n) {
  if (n === 0) return 1
  if (n === 1) return x
  if (n < 0) {
    return 1 / myPow(x, -n)
  }
  if (n % 2 === 0) {
    return myPow(x, n/2) ** 2
  } else {
    return (myPow(x, (n-1) / 2) ** 2) * x
  }
};
```

时间复杂度 O(log~2~n)

空间复杂度 O(1)
